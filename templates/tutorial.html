Steps to transfer songs from apple music to spotify

  • You'll need a json file containing the songs. You can request your personal data from apple, extract it and get the json file containing the songs in your library.
    How to request Your personal data from apple

  • Next step is to upload said file into Am2Spotify and go back to import url then wait for it to search and load the songs from the file. This step might take a while for json files with lots of songs.

  • Select the songs you wish to import and it'll create a new playlist containing the imported songs.

Steps to transfer songs between playlists

  • First step is to select the source playlist then the target playlist.

  • You'll be redirected to the transfer page where you select songs you wish to import.

  • Select either copy or move. Copy keeps the source playlist, while transfer deletes it.
